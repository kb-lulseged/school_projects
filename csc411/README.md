# README
A fully-connected neural network built from scratch to predict BRDF reflectance values. 

## Code
- `brdf_nn.py` - neural network implementation. 
- `BRDFRead.cpp` - Modified MERL reading that outputs training data

## Data
- `black_obsidian_data.txt` - training data generated by reader (7 columns: `theta_in`, `phi_in`, `theta_out`, `phi-out`, `red`, `green`, `blue`) 

## Outputs
- `error_plot.png` - Training and test errors over epochs
- `final_weights.txt` - Trained network weights (W1, b1, W2, b2)

## Presentation
- `brdf_presentation.pptx` - slides for video presentation & brief overview


## Quick Start
- Generate training data (if not already done)
```bash
g++ BRDFRead.cpp -o brdf_reader -lm
./brdf_reader black-obsidian.brinary > black_obsidian_data.txt
```
- Train the network
```bash
python3 brdf_nn.py
```
- Program outputs error plot and final weights text file


>[!info] To experiment with different values, edit `main()` function in `brdf_nn.py`
